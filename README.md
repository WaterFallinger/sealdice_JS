# SealDice JS 插件

海豹骰自定义 JS 插件集。

## 插件列表

### 1. 自动复读机
文件：+1Version6.4.js

检测群聊中三个不同成员复读同一句话时，自动跟一句复读。每条复读链只触发一次。

### 2. 深夜梦呓
文件：Dream_talk_Version1.2.js

深夜 23:00 至次日 3:00 随机弹出劝睡文案，每群每天最多 3 次。

文案中的 <<$t玩家>> 会自动替换为触发者的群昵称。

⚠️ 文案适配作者自家骰子人设，如需使用请修改代码中的 TALK_LIST 数组为你自己的文案。

### 3. AI图书管理员
文件：AILibrarian_V2.0_AI图书管理员（替换白川汐prompt）.js

通过大语言模型（需自行配置 API Key）查询 COC7 规则书原文，AI 仅做检索引用，不代 KP 做判定。

用法：.rule <问题> 或 .ask <问题>

⚠️ 整个插件的回复文案均适配作者自家骰娘"白川汐"人设（雪女口吻），如需使用请修改代码中的所有提示文案。

## 使用方法

1. 下载 .js 文件
2. 在海豹骰 WebUI -> 插件管理 -> 上传插件
3. 启用插件即可

## 环境

- 海豹骰版本：v1.5.1+
- 运行平台：Windows
